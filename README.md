# 등대지기 (Deungdaejigi)

가족돌봄청년을 위한 익명 기반 통합 지원 플랫폼

## 👨‍👩‍👧‍👦 팀원 (Infoster)

- **팀장:** 진현섭 (프론트엔드)
- **팀원:** 지하현 (프론트엔드)
- **팀원:** 임상희 (백엔드)
- **팀원:** 최진영 (백엔드)

## 🛠️ 기술 스택

- **Frontend:** Flutter
- **Backend:** Nest.js, TypeScript
- **Database:** MongoDB, Mongoose

## 🚀 프로젝트 실행 방법

### Backend

```bash
cd backend
npm install
# .env 파일 생성 및 설정 필요
npm run start:dev
```
